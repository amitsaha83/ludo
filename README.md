# Ludo Game

Ludo is a strategy board game for two to four players.

### Depencencies

1. Node
2. Any latest browser

### Steps to deploy and run

##### Install

```
npm install
```

##### Build

```
npm run build
```

##### Run

The Game can be accessed by opening the index.html file on any browser
